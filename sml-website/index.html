Vim Syntax Highlighting for SML: Vim is a powerful and highly configurable editor. Vim is free software and part of all mainstream Linux distributions but also available for other platforms. The syntax highlighting definitions just have to be copied to the .vim/syntax/ directory of your home directory. After adding the following lines to your vimrc file, the syntax highlighting is applied automatically when files with the extensions '.sparqlify' or '.sml' are opened: 
au BufNewFile,BufRead *.sparqlify setfiletype sml
au BufNewFile,BufRead *.sml setfiletype sml
Alternatively, you can enable the highlighting explicitly, typing set syntax=sml in Vim's execution mode. 		Source Code
